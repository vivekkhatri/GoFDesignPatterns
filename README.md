# Gang of Four Design Patterns in C#

This repository provides implementations of the 23 Gang of Four (GoF) design patterns in C#.

**Design Patterns Included:**

* **Creational:**
    * Abstract Factory
    * Builder
    * Factory Method
    * Prototype
    * Singleton
* **Structural:**
    * Adapter
    * Bridge
    * Composite
    * Decorator
    * Facade
    * Flyweight
    * Proxy
* **Behavioral:**
    * Chain of Responsibility
    * Command
    * Interpreter
    * Iterator
    * Mediator
    * Memento
    * Observer
    * State
    * Strategy
    * Template Method
    * Visitor
